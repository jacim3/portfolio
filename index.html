Smart MapHack

GoogleMap Api를 이용한 Native App




▶ Description


( 2019.12.10 ~ 2020.01.10 )

'Map Api와 Notification, DB를 이용하여 사용자의 위치정보와 길 찾기를 제공하며, 사용자의 일정을 쉽게 관리해주는 어플을 만들어보면 어떨까?' 하는 생각으로부터 출발하였습니다.

사용자가 등록한 일정은 DB에 등록 후, RecylerView를 통하여 보여지며, 일정 기준에 따라 배치가 이루어지도록 구현 하였습니다.

위치정보는 GoogleMap Api와 Place Api가 사용되어, 여기서 제공하는 태그와 위치정보를 가져와 사용하며, 검색버튼 하나하나에 위치정보 태그를 할당하여 각각의 마커 생성과 삭제 기능을 작동하게 하였습니다.

Notification은 일정 등록 후 현재시간과 DB에 등록된 시간을 비교하여 울리게 되며, 터치 시 곧바로 GoogleMap 길찾기로 이동하도록 하였습니다.

Basket Battle

Web과 Native의 기능이 합쳐진 App




▶ Description


( 2020.3.10? ~ 2020.04.18? )

홈 쇼핑을 자주 이용하는 사용자들에 맞춰서, 원하는 상품을 앱에 간편하게 등록하고, 이에 대한 결정을 도와주는 과정을 '재미있게 표현할 수 있지 않을까?' 하는 생각으로부터 시작되었습니다.

Android의 WebView에서 보여지는, 홈쇼핑 몰(옥션, 위메프 등) 혹은 다른 웹 사이트에서 사용자가 원하는 항목을 어플의 등록버튼을 통하여 등록하면, 이를 Java의 JSoup라이브러리를 통하여 해당 웹 페이지의 CSS내부의 OpenGraph 태그의 정보를 가져오게 함 으로써 이를 안드로이드 내 SQ-Lite에 저장합니다.

상품 선택 기능은 Native쪽의 DB에 저장된 정보를 JSON객체로 변환하여 이를 Web이 받아서 처리하며, JQuery를 통한 알고리즘 및 JQuery-Ui를 사용하여 애니메이션을 구현하였습니다. 최종으로 처리된 결과는 다시 Web에서 Native로 전달되어 반영됩니다.
